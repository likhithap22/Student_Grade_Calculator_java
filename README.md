# Student Grade Calculator (Java)

## 📌 Description
A simple Java console application that calculates the total marks, average percentage, grade, and pass/fail status of a student based on marks entered for multiple subjects.

## ✨ Features
- Accepts student name
- Accepts number of subjects
- Validates marks (0–100)
- Calculates total marks
- Calculates average percentage
- Displays grade (A–F)
- Displays Pass/Fail status

## 🛠 Technologies Used
- Java
- VS Code
- Scanner Class

## ▶️ How to Run

Compile:

```bash
javac StudentGradeCalculator.java
```

Run:

```bash
java StudentGradeCalculator
```
